SCDEPy (Version 0.1)¶

SCDEPy is a Python package for single-cell data analysis, with a special focus on detecting appropriate probabilistic models while categorizing various featuring types of genes under a specific condition.

Note

This project is under active development. It also serves as a final project for the course APC524 (Fall 2022) at Princeton University.
